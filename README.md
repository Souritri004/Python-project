# Python-project
The ATM system is the project which is used to access their bank accounts in order to make cash withdrawals, Deposits and Checking Balance. Whenever the user needs to make cash withdraws, they can enter their pin number and it will display the amount to be withdrawn. Once their withdrawn was successful, the amount will be deducted in their account. 
The ATM will service one customer at a time. A customer will need to enter ATM username and pin both will be verified and then the customer will be able to perform one or more transactions. 
In the case of cash withdrawal, a message will be sent after the transaction has been completed or accepted. Program determines the customers pin before a transaction can proceed. The ATM will give the customer with message for each successful transaction, account, amount and available balance of the affected amount.
The customers will also be able to change their pin through ATM Machine. They need to enter the old password and then the new password. Then only they will be able to change the PIN successfully. Once the new PIN is generated, it will be shown in the screen
